MissingImage

MissingImage: "error" | "ignore" | "remove"
  • Defined in section.tsx:15

how to handle missing images

A missing image is an <img /> in sections whose src attribute isn't a key in images.

  • "error" : throws and error
  • "ignore" : do nothing, keep the <img /> element
  • "remove" : filter the <img /> element
